# win32gui: AvZ 辅助工具 GUI 插件

此插件最简化封装了 win32api 接口以实现简单的 GUI 界面,
提供了五种控件: 按键(PushButton)、复选框(CheckBox)、下拉框(ComboBox)、编辑框(Edit) 和标签(Label)，
此插件使用起来非常简单，直接查看使用示例即可
